# Installation from source

### Prerequisites

 - [Node.js](https://nodejs.org)
 - [pnpm](https://pnpm.io)
 - [wasm-pack](https://rustwasm.github.io/wasm-pack)

<Steps>

### Clone

```sh
cd .vscode/extensions
git clone https://github.com/cooolbros/vscode-vdf
```

### Install Dependencies

```sh
pnpm install
```

### Build

```bash
pnpm build
```

</Steps>
